## 🛠️ Tecnologias Utilizadas
- HTML5
- CSS3

## 🚀 Instalação
1. Clone o repositório:
   ```bash
   git clone https://github.com/JosielJrr/youtube-homepage.git
   cd youtube-homepage
   ```
2. Abra o arquivo `index.html` no seu navegador.
